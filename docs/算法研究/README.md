## 先看这三条

1. **当前 consumer 榜单仍是 curated guide，不是真正的 verifier-backed 真榜。**
2. **OHBP 不应该先做“神秘总分”，而应该先做“真实任务成功率 + 可信度闸门 + 不确定性展示”。**
3. **在单人、低预算条件下，真实性的重点应放在“证据链、方法学、硬门槛、抽样复跑”上，而不是全量高成本复核。**
